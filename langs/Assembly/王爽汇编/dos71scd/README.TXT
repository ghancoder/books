MS-DOS 7.10 Installation CD
===========================

This is MS-DOS 7.10 Installation CD, which includes MS-DOS 7.10 Full Version and some Add-Ons.

Please burn the CD ISO image (DOS71CD.ISO) to a CD-R/RW disc, or use it as an emulated CD disc in a virtual machine (such as VMWare and Virtual PC), then boot your system from this Installation CD in order to install MS-DOS 7.10.

For more information, please come to:

http://newdos.yginfo.net/msdos71/index.htm
